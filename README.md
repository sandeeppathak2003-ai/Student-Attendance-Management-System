# Student Attendance Management System (C++)

This is a console-based Student Attendance Management System developed in C++.

## Features
- Update daily attendance
- Calculate attendance percentage
- Generate attendance report
- File handling for storing attendance data

## Technologies Used
- C++
- File Handling
- Object-Oriented Programming

## How to Run

Compile the program:

g++ attendance.cpp -o attendance

Run the program:

./attendance
